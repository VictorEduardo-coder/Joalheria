# 💍 Joalheria - Site Institucional

Este projeto consiste no desenvolvimento de um site focado na apresentação de produtos de uma joalheria. O objetivo foi criar uma interface limpa, elegante e responsiva para exibir catálogos e informações de contato.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as tecnologias base do desenvolvimento web:

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização, layout e design responsivo.

## ⚙️ Funcionalidades

- **Design Responsivo:** O site se adapta a diferentes tamanhos de tela (desktop, tablet e mobile).
- **Catálogo de Produtos:** Exposição visual das joias.
- **Formulário de contato:** Formulário para envio de mensagens.
